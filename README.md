# REPOSITÓRIO DA DISCIPLINA DE FUNDAMENTOS DE INTELIGÊNCIA ARTIFICIAL

As aulas foram organizadas de forma progressiva, cobrindo desde conceitos básicos até aplicações reais:

🔹 1. Introdução à Inteligência Artificial
Conceitos fundamentais
Tipos de IA
Aplicações no mundo real

🔹 2. Chatbots Conversacionais
Criação de chatbots com LLMs
Integração com APIs (ex: Groq/OpenAI)
Interface interativa com Streamlit

🔹 3. Chat com PDF (RAG)
Upload e leitura de documentos
Extração de informações com IA
Construção de sistemas de perguntas e respostas

🔹 4. Visão Computacional
Processamento de imagens
Detecção e classificação
Aplicações práticas

🔹 5. Ciência de Dados
Análise exploratória
Manipulação de dados com Pandas
Visualização de dados

🔹 6. Machine Learning
Criação de modelos preditivos
Treinamento e avaliação

##  Tecnologias Utilizadas

Este projeto utiliza um conjunto de ferramentas modernas para desenvolvimento de aplicações em Inteligência Artificial:

- 🐍 **Python** — Linguagem principal para desenvolvimento
- 📊 **Pandas, NumPy e Scikit-learn** — Análise de dados e Machine Learning
- 🎨 **Streamlit** — Criação de interfaces interativas
- 🔗 **APIs de IA (LLMs)** — Integração com modelos de linguagem
- ⚙️ **Airflow** *(em desenvolvimento)* — Orquestração de pipelines
- 🐳 **Docker** *(em desenvolvimento)* — Containerização da aplicação
